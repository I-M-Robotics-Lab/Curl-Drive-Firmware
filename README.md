# Curl-Drive Firmware: BLDC Motor Controller

This repository contains the firmware and source code of the brushless motor FOC controller **Curl-Drive** developed by I'M Robotics Lab. 

It is still under construction. Check issues for the latest dev progress. 

We welcome contributions from anyone! To get involved or ask questions, join our Discord server.
